# 🚀 ResumeAI — AI Resume Builder Landing Page

A modern, responsive landing page for **ResumeAI**, an AI-powered resume builder that helps users create job-winning resumes in minutes.

---

## ✨ Features

- 🎨 Clean Neumorphism UI design
- 📱 Fully responsive (mobile, tablet, desktop)
- ⚡ Smooth scrolling & animations
- 📊 Pricing toggle (Monthly / Annual)
- ❓ FAQ accordion interaction
- 📬 Newsletter form with validation
- 📌 Sticky navbar with active link highlighting

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Neumorphism Design System)
- Vanilla JavaScript (ES6)
